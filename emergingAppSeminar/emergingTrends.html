DARK MODE

Dark mode was introduced by Microsoft in 2016, then in 2018 Apple followed the concept (which made it very popular). It reduces the light produced by a screen by displaying dark colors, such as a darker colored background with light colored text. Using dark mode on a device helps reduce eye strain, while still showing contrast between elements. Many designers all over the world, including myself, have started copying this style. It gives a clean and clear look, while also seeming to give a more serious tone.

One thing to note as a designer, it is best not to use pure black (and even pure white) when styling a dark mode project. It is better to use a darker grey or a lighter black when making content for dark mode. This helps readability and reduces the strain on the eyes. To add something other than just dark solid colors, darker colored gradients have become popular to pair with dark mode. It is also a good idea to give the user the option to switch from dark to light modes; it is more of a preference after all.

BENTO GRIDS

Inspired by Apple, bento grids are grid style layout of boxes containing the content of a website. These grids gave a little more life to a website that is mostly informational. Bento grid can make websites look organized with a modern feel. They are also easy to use and create. The downside is that they can easily become cluttered which can overwhelm the user. The concept of the bento grid is not new, but it does seem to be a trend for 2024. In the more recent days, the bento grid is being made to be more animated and not seeming so flat.

NO-CODE TOOLS

It seems self-explanatory why no-code tools would be on this list. No-code design tools allow us to build software without writing code. The creator does not need to know how to code much in order to build a website on tools like these, it is more about the vision. Using these no-code tools can be cost effective because it usually takes more than one person to build an entire website. With a tool like this, one person can do the whole thing. (There are still options to make it a collaborative project.) It will also help with development time and they are usually very user user-friendly. In 2020 the no-code market was worth $13.2 billion and is expected to reach $45.2 billion by the year 2025.
